Where there is an Android phone, there are reasons to root it.

Here I took a note how I rootedmy HTC Sensation (A.K.A G14):

  1. Find the S/N on battery, and find HBoot version by holding "Volumn down" and long press power down.
     
  2. Install modified HTC drive from: http://downloads.unrevoked.com/HTCDriver3.0.0.007.exe
  3. Go to http://revolutionary.io/, get the beta key, run it.
  4. S-Off OK
  5. Put PG58IMG.zip under the root location of the SD card.
  6. Reboot to bootloader/HBOOT screen
  7. In bootloader it will prompt to update, press "Volumn Up" to select update
